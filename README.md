SVG
Description
This project is command line application that was made through the use of node.js and inquirer library.  The SVG project was created to let the user make an .svg file by having them answer prompts the it uses those answers to create the desired SVG logo. Therefore helping users solve the problem of having to hire a graphic designer whenever they need a simple logo for their projects.
